Experiment-9: Numerical integration
Overview:

This experiment focuses on applying numerical methods to approximate derivatives and definite integrals using MATLAB. The objective is to understand how these techniques work when analytical solutions are difficult and to evaluate their accuracy.

Methodology:
Numerical Differentiation:
Conversion of uneven data into equal intervals
Implementation of forward difference method

Numerical Integration:
Application of Trapezoidal Rule
Application of Simpson’s methods
Comparison of numerical results with analytical solutions
Analysis of effect of step size and number of intervals

Results:
Forward difference method showed higher error at initial points
Accuracy of derivative improved for subsequent values
Trapezoidal rule accuracy increased with more intervals
Simpson’s methods provided better approximation than trapezoidal rule
Higher-order methods and finer intervals resulted in improved accuracy
Observed that numerical methods always introduce some approximation error
